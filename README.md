# Eumdi Parent Planner

성남시 기준 예비 부모가 임신 확인부터 출산 후 24개월까지 필요한 체크리스트, 지원금, 준비물, 병원/조리원 비교, 가족 메모를 한 화면에서 관리하는 정적 웹앱입니다.

## 주요 기능

- 임신 주차와 출산 후 개월 수를 이어서 보는 타임라인
- 여러 카테고리 필터와 직접 추가 가능한 맞춤 카테고리
- 주차별 체크리스트, 지원금 신청, 출산 전/출산 후 준비 항목 관리
- 병원과 산후조리원 비교표
- 출산예정일, 태명, 연락처, 예산 같은 가족 정보 저장
- 브라우저 `localStorage` 기반 체크 상태 저장

## 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

GitHub Pages에 배포할 경우 저장소 루트의 `index.html`을 그대로 서비스하면 됩니다.

## 자료 기준

초기 데이터는 2026년 성남시 예비 부모 로드맵/매뉴얼 문서를 바탕으로 정리했습니다. 지원금과 신청 조건은 정책 변경 가능성이 있으므로 실제 신청 전에는 복지로, 성남시, 관할 보건소 안내를 확인해야 합니다.
